# ReadyWrite

ReadyWrite is a voice-first writing platform designed to help authors, preachers, coaches, and creators turn spoken ideas into structured, publish-ready writing.

## What it does
- Transcribes voice and audio into clean text
- Preserves the author’s original voice and tone
- Automatically structures chapters and sections
- Exports to Word and Google Docs formats

## Who it’s for
- Authors and writers
- Preachers and ministry leaders
- Coaches and educators
- Content creators

## Tech Stack
- Next.js
- React
- Framer Motion
- Lucide Icons
- Deployed on Vercel

## Status
Early-stage MVP / Landing page.

## Roadmap
- User authentication
- Audio upload + transcription
- Chapter auto-generation
- Export and publishing tools
- Subscription billing
